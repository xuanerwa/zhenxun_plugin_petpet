# zhenxun_plugin_petpet

zhenxun_bot 头像表情包插件

移植自[nonebot-plugin-petpet](https://github.com/noneplugin/nonebot-plugin-petpet)

## 插件安装

1. 将本插件文件放到自定义插件目录下，放到zhenxun_bot/plugins下也行(不推荐，更新会替换这个目录)

2. 安装相关依赖

   ```
   pip3 install -r requirements.txt
   ```

### 使用

- 指令 + @user，如： 爬 @小Q
- 指令 + qq号，如：爬 123456
- 指令 + 自己，如：爬 自己
- 指令 + 图片，如：爬 [图片]
